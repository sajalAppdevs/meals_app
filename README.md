# MealsApp

## Overview
MealsApp is a feature-rich Flutter application that helps users discover and manage meal recipes. Built with Flutter's latest navigation and screen management features, this app provides an intuitive interface for browsing meal categories, accessing detailed recipes, and personalizing your cooking experience.

## Features
- **Category-based Meal Browsing**: Browse meals organized by categories for easy navigation
- **Detailed Recipes**: Access comprehensive recipes with step-by-step cooking instructions
- **Favorites System**: Mark and save your favorite meals for quick access
- **Filtering Options**: Customize meal suggestions based on dietary preferences
- **Responsive UI**: Beautiful and responsive design that works across different screen sizes

## Getting Started

### Prerequisites
- Flutter SDK (>=2.17.3 <3.0.0)
- Dart SDK
- Android Studio / VS Code with Flutter extensions

### Installation
1. Clone the repository
```bash
git clone https://github.com/yourusername/meals_app.git
```

2. Navigate to project directory
```bash
cd meals_app
```

3. Install dependencies
```bash
flutter pub get
```

4. Run the app
```bash
flutter run
```

## Dependencies
- Flutter SDK
- cupertino_icons: ^1.0.2
- flutter_lints: ^2.0.0 (dev dependency)
